# lazy-album 懒人相册(WIP)

> [!WARNING]
> This plugin is currently under active development. The documentation and features are not finalized yet. 
> 
> 本插件正处于开发阶段，功能与文档尚未最终确定。

一个obsidian插件，可以把图片以瀑布流形式呈现，并支持混合多种不同的图片来源,如：
- 网络图片链接
- 库内图片路径
- 库内图片文件夹相对路径，会自动加载所有图片。

其他功能包括
- columns: 改变栏数  
- exclude: 选择性剔除文件夹中不想展示的图片。  
- gap: 改变图片间距  
- perpage: 拟定每页最多展示的图片数，分页展示。  
- 通过" | "为插入的单张图标注caption并在鼠标悬停时显示（否则默认网络图片为无，本地图片为文件名）
