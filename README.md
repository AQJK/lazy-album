# lazy-album 懒人相册
一个obsidian插件，可以把图片以瀑布流形式呈现，并支持混合展示多种不同的图片来源,如：
- 网络图片链接
- 库内图片路径
- 库内图片文件夹相对路径，会自动加载所有图片。
并可以选择性剔除文件夹中的一些文件。


如果您刚开始使用obsidian，强烈推荐先了解obsidian的代码块与模板功能。  

启用插件后，您可以把下面这段示例代码直接粘贴到你的文本中。当您替换掉代码块中的图片来源后，应该就可以展示出图片了。
